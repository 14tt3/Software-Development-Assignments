# Software-Development-Assignments

magic8ball description: It’s basically a little Magic 8 Ball app using Flet. You type a yes/no question, hit the button, and it gives out a random answer. The answer changes color depending on if it’s positive, negative, or neutral. 

number_guesser description: This is a number guessing game where the computer picks a random number from 1–50 and you try to guess it. You can choose between limited tries or unlimited. The game gives you hints like “too high,” “too low,” or “warmer” if you’re close. It also keeps track of all your guesses and shows them at the end or when you win in unlimited mode.
